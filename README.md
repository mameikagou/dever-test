# Dever-新一代并发vibe coding编程辅助应用

#### 前置条件: 
- 需要安装 [Node.js](https://nodejs.org/) 环境
- codex-cli 工具（`npm -g install @openai/codex`）

#### 演示视频：
- [Dever 使用演示视频](https://www.bilibili.com/video/BV17orXBJEYi)

#### 演示平台：
- 魔搭：https://modelscope.cn/studios/xiaoyumodel/dever/summary

#### 安装位置：
本页面右侧release点击

#### mac版本安装问题：
如果提示已经损坏，是苹果的限制问题：请在安装后用命令行运行以下命令：
```
xattr -dr com.apple.quarantine /Applications/dever-desktop.app
```
